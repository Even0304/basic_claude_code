# 🚀 kimi-code 快速参考

## 三种使用方式

### 方式 1️⃣：看演示（最简单）
```bash
kimi-code demo parallel
```
看 4 个任务同时执行，展示并行能力。

### 方式 2️⃣：交互模式（推荐）
```bash
kimi-code
```
然后输入问题，例如：
```
> 统计这个项目有多少个 Python 文件
> 找出所有 TODO 注释
> 并行分析：文件数、代码行数、依赖、TODO
> /help      # 查看所有命令
> /exit      # 退出
```

### 方式 3️⃣：单次命令
```bash
kimi-code run "统计 Python 文件个数"
```

## 并行执行技巧

### ❌ 慢速方式（顺序执行）
```
> 统计文件个数        (10秒)
> 计算代码行数        (10秒)
> 列出依赖            (10秒)
> 找出 TODO          (10秒)
总耗时: 40秒
```

### ✅ 快速方式（并行执行）
```
> 并行做这些：统计文件、计算行数、列出依赖、找 TODO
总耗时: 10秒 (快4倍！)
```

### 触发并行的关键词
- "并行"
- "同时"
- "并发"
- "同步执行"

## 实际例子

### 例子 1：快速分析项目
```bash
kimi-code run "分析项目：统计 Python 文件、计算总行数、列出依赖"
```

### 例子 2：找出所有问题
```bash
kimi-code run "找出所有 TODO、FIXME、BUG 注释"
```

### 例子 3：并行分析（推荐）
```bash
kimi-code run "并行执行：
  1. 统计 Python 文件数
  2. 计算代码总行数
  3. 列出项目依赖
  4. 找出所有 TODO 注释"
```

### 例子 4：交互式编程
```bash
kimi-code
> 创建 hello.py，打印 Hello World
> 检查文件是否创建成功
> 编辑文件，把 Hello 改成 Hi
> /exit
```
